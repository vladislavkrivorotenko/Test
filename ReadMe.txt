Подготовка к запуску
1. Зайдите в файл "script.ipynb".
2. Установите в bd_name новое зачение в формате *.db.
3. Измените тип окна после заголовка "Альтернативный скрипт с фильтрацией истории оконными функциями" на значение "code".
4. Замените везде, где написано "row_number" на "dense_rank".
5. Перейдите в первое окно ноутбука, содержащее заголовок "Подключаем необходимые библиотеки".
6. Запустите последовательное исполнение всех окон ниже. Cell -> Run All Below.
